# Stock News Alert System

A Python-based alert system that tracks stock price changes and fetches related financial news.

## Features
- Monitors one or more stock symbols
- Retrieves daily stock price data
- Fetches related news headlines per stock
- Generates alerts based on price movement thresholds

## Tech Stack
- Python
- REST APIs
- JSON

## How It Works
1. Fetches stock price data from financial APIs  
2. Calculates percentage change per trading session  
3. Retrieves recent news articles for the stock  
4. Triggers alerts when significant changes occur  

## Usage
- Configure stock symbols and thresholds
- Execute the script to receive alerts

## Future Improvements
- Add historical trend analysis
- Integrate push or email notifications

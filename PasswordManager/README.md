# Password Manager

A simple Python-based password manager for securely storing and retrieving credentials.

## Features
- Stores credentials for multiple services
- Supports add, search, and retrieve operations
- Prevents duplicate or invalid entries
- Uses structured file-based storage

## Tech Stack
- Python
- File Handling

## How It Works
1. Accepts user input for service credentials  
2. Stores data in a structured local file  
3. Allows searching and retrieval by service name  

## Usage
- Run the script and follow the prompts
- Add or retrieve stored credentials as needed

- Encrypt stored passwords

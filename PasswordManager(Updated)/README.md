# Password Manager

Password Manager is a Python-based desktop application that allows users to generate, store, and retrieve login credentials for multiple websites using a graphical user interface.

---

## Features

- Generates strong random passwords using letters, numbers, and symbols  
- Stores website credentials locally in a structured JSON file  
- Allows searching saved credentials by website name  
- Copies generated passwords directly to the clipboard  
- Provides input validation and user-friendly error messages  

---

## Tech Stack

- Python  
- Tkinter  
- JSON  
- pyperclip  

---

## How It Works

1. Generates secure passwords using random character combinations  
2. Accepts website, email/username, and password inputs via GUI  
3. Stores credentials in a local JSON file  
4. Retrieves saved credentials using website-based search  

---

## Usage

- Run the Python script to launch the application  
- Enter website and email/username details  
- Generate or manually enter a password  
- Click **Add** to save credentials or **Search** to retrieve them
